# imessage — MERN Real-Time Chat App

A full-stack real-time messaging app with media sharing, online presence, and theme customization.

**Stack**: Express 5 · Mongoose · Socket.IO · Clerk (auth) · ImageKit (media) · React 19 · Vite 8 · Tailwind 4 · Zustand · HeroUI

## Structure

```
backend/    Express API, Socket.IO server, MongoDB models
frontend/   React SPA (Vite), Zustand stores, HeroUI components
```

See `AGENTS.md` for commands, architecture notes, and conventions.
